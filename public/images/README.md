# Imágenes del Proyecto

Esta carpeta contiene todas las imágenes locales del sitio web.

## Estructura de Carpetas

- `hero/` - Imagen de fondo para la sección principal (Hero)
- `about/` - Imágenes para la sección "About Us"
- `projects/` - Imágenes de los proyectos

## Cómo Agregar Imágenes

### Hero Section
Coloca una imagen en `hero/hero-background.jpg` (o .png, .webp)
- Recomendado: 1800x1200px o más grande
- Formato: JPG, PNG o WebP

### About Section
Coloca dos imágenes en `about/`:
- `about-main.jpg` - Imagen principal (900x675px recomendado)
- `about-inset.jpg` - Imagen pequeña (400x400px recomendado)

### Projects
Para cada proyecto, crea una carpeta con el ID del proyecto:
- `projects/1/` - Proyecto 1 (industrial-slab-warehouse)
- `projects/2/` - Proyecto 2 (residential-driveway-polished)
- etc.

Dentro de cada carpeta de proyecto, agrega:
- `cover.jpg` - Imagen de portada
- `1.jpg`, `2.jpg`, `3.jpg` - Imágenes adicionales de la galería

## Formatos Soportados
- JPG/JPEG
- PNG
- WebP (recomendado para mejor rendimiento)

## Tamaños Recomendados
- Hero: 1800x1200px o más grande
- About main: 900x675px
- About inset: 400x400px
- Project cover: 1200x800px
- Project gallery: 1200x800px
